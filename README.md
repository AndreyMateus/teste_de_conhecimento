**<p>.NET - 6</p>**
**<p>SQL SERVER</p>**
***

## Como Baixar e rodar o projeto
1. Baixe ou clone o projeto em qualquer lugar de sua máquina
2. abra a solution do projeto em qualquer IDE/EDITOR
3. Vá no `AppDBContext` e altere a `Connectrion String` pela sua.
4. vá até a pasta do Projeto vi CLI(Terminal)
5. execute um `dotnet restore`
6. Execute um `dotnet ef database update` (para subir a migration e criar o banco e as tabelas)
7. execute um `dotnet run`
